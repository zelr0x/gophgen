Go PlaceHolder Generator

Create placeholders and identicons right in the html img tag!

There are a few rules:

  1. Specify identicon with "i" if you don't want a simple rectangle
  2. Specify width or WxH. Other parameters can't be specified without width. If omitted, default of 64x64 is used
  3. Specify background color first if you want to specify foreground color
  4. Seed data can be specified after "d="
  5. Preserve order

Examples

<img src="http://localhost:8080/i/64">
<img src="http://localhost:8080/i">
<img src="http://localhost:8080/i/64/53f">
<img src="localhost:8080/i/64/099b8c/bc3a1a/d=UserName">   <-- big one
<img src="http://localhost:8080/i/64/53f/fuchsia">
<img src="http://localhost:8080/89ff/d=UserName120">   <-- random square, subsequent parameters are ignored because width is omitted
<img src="http://localhost:8080/128x32/dark-goldenrod/d=UserName120">


API

origin[/i]/width[xHeight][[[/bgColor][/fgColor]][/extension][d=data]]

  1. "i" - optional. Flag to generate identicon. String, no quotes
  2. Size:
    1. Width - required. Integer (1-4096)
    2. Height - optional. Add x as a delimeter (e.g. 128x64) between width and height. Otherwise height is the same as width
  3. Color:
    1. Background Color - optional. Hexadecimal number like #099b8c or a string like red
    2. Foreground Color - optional. Should be used only with preceding background color
  4. Extension - optional. Can be png, jpeg or gif
  5. Data - optional, should be prepended with d=. Data is used as a seed for identicon generator, which is useful for creating avatars
